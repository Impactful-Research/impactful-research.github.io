# impactful-research.github.io — Project Instructions

## What This Repo Is

Organization blog for Impactful Research. Built with Jekyll using the Minimal Mistakes theme, hosted on GitHub Pages at https://impactful-research.github.io.

---

## Current Work and Write Access

Write access:
- **Editable:** `_pages/`, `_posts/`, `_data/`, `_includes/`, `assets/` — page content, posts, and site data
- **Editable:** `_config.yml` — site configuration
- **Read-only:** `_layouts/`, `_sass/` — theme overrides (edit only if needed)

---

## Project Structure

```
impactful-research.github.io/
├── _config.yml         # Site settings (title, URL, theme skin, author info)
├── _config_dev.yml     # Dev overrides
├── _pages/             # Site pages (index.md, about.md, year-archive.md)
├── _posts/             # Blog posts (YYYY-MM-DD-slug.md)
├── _data/              # Structured data (navigation, author info)
├── _includes/          # HTML partials / overrides
├── _layouts/           # Layout templates
├── _sass/              # SCSS theme overrides
├── assets/             # Static files (CSS, JS, images, PDFs)
└── Gemfile             # Ruby gem dependencies (Jekyll + plugins)
```

See `README.md` for local preview setup and key files to edit.
